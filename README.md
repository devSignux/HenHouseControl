What I need:
- 1 x Onion Omega
- 1 x Onion Omega Expansion Dock
- 1 x Usb-cable (optional)
- 1 x 12V 30WH Accumulator
- 1 x 50W Sorarpanel
- 1 x solar load control
- 2 x 1kOhm Resistor
- 2 x 10kOhm Resistor
- 2 x Magnetic contact ( e.g. ABUS MK2000W )
- 1 x Gearbox Single Ratio 148:1 (MFA)
- 1 x Motor Driver L293D
- n x Power cable
